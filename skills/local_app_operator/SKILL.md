# Local App Operator

## Trigger

- Use when the task needs local program search, whitelist control, path selection, or app launching.

## Typical Work

- Locate the target executable and verify whether it is allowed.
- Save favorites or update whitelist status before use.

## Outputs

- Program candidates, whitelist decisions, and launch records.
