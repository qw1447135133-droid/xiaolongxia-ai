## 🐛 原始问题

**症状：** 双击 `小龙虾AI团队.exe` 会打开超级多的窗口

**影响：** 严重 - 应用无法正常使用

---

## 🔍 根本原因

### 1. 缺少单实例锁
- 没有使用 `app.requestSingleInstanceLock()`
- 允许多个应用实例同时运行
- 每次点击都创建新窗口

### 2. 窗口重复创建
- `createMainWindow()` 没有检查窗口是否已存在
- 可能创建多个主窗口

### 3. 文件路径错误
- 生产环境路径：`__dirname/out/index.html`（错误）
- 正确路径：`__dirname/../out/index.html`
- 导致无法加载 HTML，触发重复启动

### 4. Next.js 配置冲突
- 没有配置静态导出（`output: 'export'`）
- API 路由与静态导出不兼容
- 构建失败

---

## 🛠️ 修复方案

### ✅ 修复 1：添加单实例锁

**文件：** `electron/main.cjs`

```javascript
// 防止多实例启动
const gotTheLock = app.requestSingleInstanceLock();
if (!gotTheLock) {
  console.log('[main] Another instance is already running, quitting...');
  app.quit();
  process.exit(0);
}

// 处理第二个实例尝试启动
app.on('second-instance', () => {
  if (mainWindow) {
    if (mainWindow.isMinimized()) mainWindow.restore();
    mainWindow.focus();
  }
});
```

**效果：**
- ✅ 只允许一个应用实例
- ✅ 再次点击会聚焦到已有窗口

### ✅ 修复 2：防止窗口重复创建

**文件：** `electron/main.cjs`

```javascript
function createMainWindow() {
  // 防止重复创建
  if (mainWindow && !mainWindow.isDestroyed()) {
    console.log('[main] Window already exists, focusing...');
    mainWindow.focus();
    return;
  }
  // ... 创建窗口逻辑
}
```

**效果：**
- ✅ 检查窗口是否已存在
- ✅ 避免创建多个主窗口

### ✅ 修复 3：修正文件路径

**文件：** `electron/main.cjs`

```javascript
const url = isDev()
  ? 'http://localhost:3000'
  : `file://${path.join(__dirname, '..', 'out', 'index.html')}`;
```

**效果：**
- ✅ 正确定位到打包后的 HTML 文件
- ✅ 避免 404 错误

### ✅ 修复 4：配置静态导出

**文件：** `next.config.js`

```javascript
const nextConfig = {
  reactStrictMode: true,
  ...(process.env.ELECTRON_BUILD === 'true' && {
    output: 'export',
    distDir: 'out',
    images: {
      unoptimized: true,
    },
  }),
};
```

**效果：**
- ✅ 条件启用静态导出
- ✅ 仅在 Electron 打包时生效

### ✅ 修复 5：移除 API 路由

**操作：** 将 `src/app/api` 移动到 `src/api.backup`

**原因：**
- API 路由与静态导出不兼容
- Electron 版本使用 WebSocket 通信
- API 路由不需要

**效果：**
- ✅ 解决构建冲突
- ✅ 简化架构

### ✅ 修复 6：改进窗口关闭行为

**文件：** `electron/main.cjs`

```javascript
let isQuitting = false;

mainWindow.on('close', (e) => {
  if (!isQuitting) {
    e.preventDefault();
    mainWindow.hide();
  }
});

app.on('before-quit', () => {
  isQuitting = true;
});
```

**效果：**
- ✅ 关闭窗口时隐藏而不是退出
- ✅ 从任务栏退出才真正关闭
