# Incident Response

## Trigger

- Use when an alert, outage, or failure needs structured response handling.

## Typical Work

- Judge severity and organize the response path.
- Prepare recovery communication and incident notes.

## Outputs

- Alert summaries, response actions, and recovery messaging.
