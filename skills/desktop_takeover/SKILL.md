# Desktop Takeover

## Trigger

- Use when a desktop app or UI task needs screenshot targeting, clicks, typing, or human-assisted continuation.

## Typical Work

- Capture the current screen and locate the target.
- Continue with pointer or keyboard actions and preserve recovery context.

## Outputs

- Desktop action traces, screenshot evidence, and recovery hints.
