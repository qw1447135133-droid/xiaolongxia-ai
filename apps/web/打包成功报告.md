## 📝 注意事项

### 系统要求
- Windows 10/11 (64位)
- 内存：4GB 以上
- 磁盘：500MB 以上

### 网络要求
- 首次运行需要联网配置 API Key
- 运行时需要访问 AI 服务（OpenAI 或 SiliconFlow）

### 端口占用
- WebSocket 服务器使用端口 3001
- 确保该端口未被占用

## 🐛 故障排除

### 问题：无法下载 Electron
**解决：** 已配置国内镜像（npmmirror.com），检查 `.npmrc` 文件

### 问题：应用无法启动
**解决：** 检查是否安装了 Visual C++ Redistributable

### 问题：WebSocket 连接失败
**解决：** 检查端口 3001 是否被占用，查看防火墙设置

### 问题：AI 服务无响应
**解决：** 验证 API Key 是否正确，检查网络连接
